Évaluation quotidienne de votre état

Évaluez votre état avant et après les exercices de TCC. Des évaluations régulières aident à suivre l’évolution et à mieux comprendre vos progrès.

Vous pouvez choisir le format qui vous convient. Nous proposons deux options :

  • Commencez par une évaluation le matin. Dans la journée, faites des exercices, menez des expériences ou travaillez avec un thérapeute. Le soir, refaites une évaluation pour observer les changements.
  • Ou bien évaluez votre état juste avant de commencer à travailler dans l’application ou avant une séance avec un psychologue, puis faites une deuxième évaluation juste après avoir terminé les exercices.

    • Les deux méthodes sont tout aussi utiles. Choisissez celle qui vous convient le mieux.
