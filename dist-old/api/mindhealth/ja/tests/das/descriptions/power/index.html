「全能感」の項目は、自分を世界の中心のように感じていないか、あるいは自分に関係のない出来事にまで責任を抱え込んでしまっていないかを評価します。マイナスの結果は、本来自分の影響の及ばない出来事まで「自分のせいだ」と考える認知のゆがみが強いことを示します。他人の問題行動まで背負い込み、自責感や自己非難で身動きが取れなくなりやすい状態です.

一方、プラスの結果は、「自分は宇宙の中心ではないし、すべてのことに責任を負う必要はない」と理解していることを示します。これは冷たさではなく、自他の境界を健全に保つ視点であり、対人関係をむしろ安定させる土台になります.

その人は他者を対等なパートナーとして扱い、意見の違いを恐れません。相手を操作しようとしないため、周囲は安心して関わることができ、その人の考えにも耳を傾けやすくなります.

自ら権力を求めないにもかかわらず、自然と信頼を集め、責任ある役割を任されることも多くなります。家族や親しい人との関係も、依存ではなく相互性に基づいており、支配しようとしない姿勢が、結果として愛情と尊敬を引き寄せるのです.
