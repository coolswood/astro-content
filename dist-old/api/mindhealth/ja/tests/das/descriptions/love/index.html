「愛情」の項目は、他者からの愛にどれだけ依存せずにいられるかを測ります。プラスの結果は、愛情を大切にしながらも、それだけが人生の関心事ではないという、バランスの取れた姿勢を示しています。

その人にとって愛は大切な要素ではあるものの、幸福や自己尊重の唯一の条件ではありません。そのため、自分自身を大切にしながら、人生のさまざまな側面に関心を向けることができ、周囲からも魅力的な存在として映ります。

一方でマイナスの結果は、幸せや生きていくためには愛情が欠かせないと感じ、他者からの愛に強く依存している傾向を示します。

数値が-10に近づくほど、愛情への依存度が高まり、「見捨てられること」への恐れから従属的な立場に身を置きやすくなります。その結果、自己への敬意を失いやすく、周囲からは負担の大きい存在として受け取られることもあります。

拒絶や見放された経験をきっかけに、感情が爆発しやすくなったり、注目や同情を得ようとして、相手を操作するような行動に出ることもあります。愛情や関心を強く求めすぎる姿勢は、人々を遠ざけてしまい、逆に孤独感を深めてしまうことがあります。
