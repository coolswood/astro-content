「自立性」の項目は、自分の内側から喜びや満足を見いだせるかどうかを測ります。プラスの結果は、自分の気分や感情が、自分自身の考え方やものの見方によって形づくられていることを理解している状態を示します。その人は、自分の感情には自分が責任を持っていることを自覚しています.

そのため、「自分は完全にひとりだ」とは感じにくく、自分の心の中に広がる世界を探求し、味わうことができます。内面の世界を豊かに感じられることで、一人の時間も意味のあるものとして過ごせます.

一方、マイナスの結果は、喜びや満足感の源が外側の世界にしかないと感じている傾向を示します。この場合、外部の出来事を自分でコントロールすることは難しいため、気分は環境や他人の反応に大きく左右されてしまい、心の安定を保つことが難しくなりがちです.
