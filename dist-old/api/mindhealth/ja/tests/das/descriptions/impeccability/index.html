「完璧主義」の項目は、あらゆる面での完全さをどれほど求めるかを測ります。マイナスの結果のとき、その人は自分に絶対的な完璧さを求め、ミスは許されないもの、失敗は大惨事、つらい感情さえ「終わり」のように感じやすい傾向があります.

「常に完璧に考え、行動しなければならない」と考えるあまり、少しでも理想から外れると、自分を完全な失敗だと感じてしまいます。理想へ向かって走り続けるものの、ひとつ達成するとすぐに次の目標が現れ、望んでいた充足感が得られないまま、人生全体から楽しさや喜びが失われていきます.

プラスの結果は、現実的な目標を設定し、その過程そのものからも楽しさや達成感を得られることを示します。結果だけでなくプロセスも大切にし、「何もかも完璧でなくてよい」「失敗も学びのきっかけになりうる」と考えられる柔軟さを持っています.

皮肉なことに、こうした人のほうが、強い完璧主義の人よりも成果を上げることが少なくありません。細部にこだわりすぎて動けなくなることが少ないため、人生全体が凍りついた氷のようではなく、川の流れのようにしなやかに進んでいくのです.
