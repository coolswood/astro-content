マイナスの結果は、成功や愛情、幸福などを「絶対に欠かせないもの」とみなし、自分にはそれらが当然与えられるべきだと感じている傾向を示します。自分の資質や実績があるのだから、周囲や世界は好意的に応じるべきだ、と期待してしまうのです.

しかし、現実がそうした期待どおりに進むことは多くなく、そのギャップが抑うつやさまざまなストレス反応を生みやすくなります。エネルギーの多くが不満や後悔、怒り、悲しみなどに費やされ、人生全体が苦く、重たいものとして感じられやすくなります.

その一方で、実際に問題を解決する行動にはなかなか移らず、不満を口にしながらも状況は変わらない、という悪循環に陥りやすくなります。その結果、人生への満足感はさらに低下していきます.

プラスの結果は、「自分は何か特別な運命のために生まれてきたわけではない」と理解しつつ、自分なりの目標に向かって努力できるスタンスを示します。目標達成を目指しながらも、世界が自分の願いどおりに動くはずだとは考えません.

望みが叶わなかったときも、それを人生の破局ではなく「残念な出来事のひとつ」として受け止めることができます。世の中がいつも公平で完璧であるとは期待せず、それでも粘り強さと回復力を武器に前に進んでいくため、長い目で見ると他の人たちを追い抜いていくことも少なくありません.
