הפריט „אישור” בוחן נטייה לקשור את הערך העצמי לדעת הסביבה. תוצאה חיובית בטווח שבין 0 ל־10 מעידה על עצמאות, תחושת ערך עצמי בריאה ויכולת להגיב באופן מאוזן לביקורת ולחוסר אישור.

תוצאה שלילית בטווח שבין ‎-10 ל־0 מצביעה על תלות באופן שבו אחרים תופסים את האדם. כאשר מבקרים אותו, הוא עלול לראות את עצמו כחסר ערך. רגישות יתר רגשית לדעות אחרים הופכת אותו לפגיע למניפולציות.
