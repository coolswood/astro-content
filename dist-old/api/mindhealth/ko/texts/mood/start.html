매일 상태 평가하기

CBT(인지행동치료) 연습을 하기 전과 후에 자신의 상태를 평가해 보세요. 꾸준히 기록하면 변화의 흐름을 알아차리고, 내가 어떻게 나아가고 있는지 이해하는 데 도움이 됩니다.

원하는 방식으로 진행하실 수 있어요. 저희는 두 가지 방법을 제안합니다:

  • 먼저 아침에 평가해 보세요. 하루 동안 연습을 하고, 실험을 진행하거나 치료사와 함께 작업한 뒤, 저녁에 다시 평가해 변화를 확인합니다.
  • 또는 앱을 사용하기 직전이나 상담(치료) 세션 전에 상태를 평가하고, 두 번째 평가는 연습을 마친 직후에 바로 진행할 수도 있어요.

    • 두 방법 모두 똑같이 유익합니다. 본인에게 가장 잘 맞는 방식을 선택하세요.
