"완벽주의" 항목은 모든 것에서 완벽을 추구하는 성향을 측정합니다. 부정적인 결과에서는 사람이 자신에게 절대적인 완벽함을 요구합니다: 실수는 용납되지 않고, 실패는 재앙으로 인식되며, 심지어 부정적인 감정조차 세상의 종말로 여깁니다.

사람은 완벽하게 행동하고 생각해야 한다고 믿으며, 그렇지 않으면 완전한 실패감을 경험합니다. 이상으로의 도피는 끝없는 과정이 됩니다: 하나의 목표를 달성하면 곧바로 다른 목표가 나타나고, 기대했던 승리의 기쁨은 결코 오지 않습니다. 비현실적인 개인 기준으로 인해 삶에서 기쁨과 즐거움이 사라집니다.

긍정적인 결과는 합리적인 목표를 설정하고 결과에만 집중하지 않고 그것을 달성하는 과정에서 즐거움을 찾는 능력을 반영합니다. 이런 사람은 모든 면에서 완벽할 필요가 없으며 실수를 두려워할 필요가 없다는 것을 인식하고, 실수를 학습과 자기 개선의 기회로 받아들입니다.

역설적이게도, 이런 사람들은 세부 사항에만 집중하지 않기 때문에 종종 그들의 "완벽한" 동료들보다 더 생산적입니다. 그들의 삶은 완벽에 집중하는 친구들의 삶을 비교할 수 있는 "영원한 얼음"과 달리 강물의 흐름과 같습니다.
