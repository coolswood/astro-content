Пункт «Схвалення» оцінює тенденцію пов'язувати самооцінку з думкою оточуючих. Позитивний результат у діапазоні від 0 до 10 вказує на те, що людина володіє незалежністю, здоровим почуттям власної гідності, адекватно реагує на критику та незадоволення.

Негативний результат від -10 до 0 свідчить про залежність людини від сприйняття її оточуючими. У разі критики така людина схильна вважати себе нікчемою. Емоційна надмірна чутливість до думок інших робить її вразливою до маніпуляцій.
