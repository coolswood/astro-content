Anslut din psykolog direkt i appen!

Nu kan din psykolog granska anteckningarna från din dagbok med automatiska tankar.

Alla anteckningar uppdateras automatiskt i hens personliga konto, och du kan alltid återkalla åtkomsten när som helst.

Hur du ansluter en psykolog:

  • Klicka på knappen "Bjud in psykolog" och skicka länken till din specialist.
  • Psykologen accepterar inbjudan genom att logga in på sitt personliga konto.
  • Klart! Ni är synkroniserade. 🎉

    • Hela processen tar inte mer än 3 minuter! 🕒
