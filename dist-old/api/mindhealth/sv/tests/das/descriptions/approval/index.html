Punkten "Godkännande" utvärderar tendensen att koppla självrespekt till andras åsikter. Ett positivt resultat i intervallet från 0 till 10 indikerar att personen har självständighet, en hälsosam självkänsla och reagerar adekvat på kritik och ogillande.

Ett negativt resultat från -10 till 0 visar att personen är beroende av hur andra uppfattar hen. Vid kritik tenderar en sådan person att se sig själv som värdelös. Emotionell överkänslighet för andras åsikter gör hen sårbar för manipulation.
