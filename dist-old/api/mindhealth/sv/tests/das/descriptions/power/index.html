Punkten "Allmakt" utvärderar personens tendens att se sig själv som universums centrum och ta ansvar för allt som händer. Ett negativt resultat indikerar en kognitiv förvrängning där personen orimligt tar på sig ansvar för händelser som inte beror på dem. Hen anklagar sig själv för andras förseelser, känner skuld och självfördömelse, vilket gör dem psykologiskt hjälplösa.

Ett positivt resultat visar däremot att personen inser att hen inte är "universums centrum" och inte bär ansvar för alla och allt. Detta tillvägagångssätt isolerar dem inte från omgivningen, utan hjälper tvärtom till att etablera hälsosamma relationer.

Hen behandlar människor som jämlika partners utan rädsla för oenighet. Rättvis behandling av andra attraherar dem, eftersom hen inte försöker kontrollera dem. Hens idéer lyssnas på eftersom hen inte tvingar sina åsikter på andra.

Hen strävar inte efter makt, men människor anförtror dem den gärna. Hens relationer med närstående bygger på ömsesidighet, inte beroende. Tack vare detta vinner hen kärlek och respekt från andra utan att försöka dominera dem.
