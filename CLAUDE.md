В случае перевода правила описаны тут для всех языков - TRANSLATION_GUIDELINES.md.

Дополнительно правила для конкретного языка:

Японский - TRANSLATION_GUIDELINES_JP.md
Украинский - TRANSLATION_GUIDELINES_UK.md
Немецкий - TRANSLATION_GUIDELINES_DE.md

Если перевод уже существует, значит внимательно проведи его контроль качества сверяясь с оригиналом (ru).

При выполнении контроля качества проверяй все файлы, не пропускай ничего.

Когда переводишь специфичные термины, выписывай их в файл правил для соответствующего языка. Чтобы по всему проекту перевод был одинаков.

Переводи аббривиатуру AI, если это принято в целевом языке.

## Общие правила

- Всегда отвечай на русском языке, если пользователь явно не попросил об обратном.
- После внесения изменений всегда запускай генерацию файлов (`npm run build`) и проверяй результат, чтобы убедиться в корректности изменений.
