В случае перевода правила описаны тут для всех языков - TRANSLATION_GUIDELINES.md.

Если перевод уже существует, значит внимательно проведи его контроль качества сверяясь с оригиналом (ru).

При выполнении контроля качества проверяй все файлы, не пропускай ничего.

Когда переводишь специфичные термины, выписывай их в файл правил для соответствующего языка. Чтобы по всему проекту перевод был одинаков.
