{{INCLUDE:common.txt}}

Стиль и Тон:
- Перевод должен быть профессиональным, но теплым и поддерживающим (сфера Care/Mental Health). Личный, уважительный подход.
- Текст должен звучать максимально нативно для носителя соответствующего языка.

- pt (Португальский)

Задачи:
1. Перевод: Переведи значения из предоставленного JSON на **ВСЕ** указанные выше языки. Обязательно обработай каждый из 18 языков.

Формат ответа:
Верни **ТОЛЬКО** JSON-объект, где:
- Ключи верхнего уровня — это **коды языков** из списка выше (например, "en", "de", "pt_BR").
- Значения — объекты с оригинальными ключами исходного JSON и переведенным текстом.
- Старайся подбирать переводы примерно соответсвующие по длине оригинальным.
- Ключи, начинающиеся с @ (например, "@key"), содержат описание и контекст для основного ключа "key". Используй их для более точного перевода, но само описание переводить не нужно. Удаляй эти информационные ключи из итогового JSON.

Пример ответа:
```json
{
  "en": {
    "greet": "Hello",
    "settings": "Settings"
  },
  "de": {
    "greet": "Hallo",
    "settings": "Einstellungen"
  }
}
```
Никаких комментариев до или после. СТРОГО только JSON.
